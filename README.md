# HMathLib

HMathLib 是一个用于实现图形学计算的数学库

## 环境配置

### 步骤 1：下载并安装 vcpkg

在命令行中执行以下命令：

```bash
git clone https://github.com/microsoft/vcpkg.git
```

### 步骤 2：安装 eigne

在命令行中执行以下命令：

```bash
vcpkg install Eigen3
```
